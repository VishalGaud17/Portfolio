Credit Score Classification

Overview : A machine learning project that classifies customer credit scores based on demographic and financial features using the K-Nearest Neighbors (KNN) algorithm. The model is optimized through cross-validation and evaluated using classification reports. It is deployed as a .pkl file and integrated into a Streamlit web app that allows users to input data and receive real-time credit score predictions.
